# statistical-regression-analysis
Comprehensive regression modeling in Python including linear, multiple, logistic, multicollinearity diagnostics, and polynomial regression using real datasets.
# Regression Modeling in Python

This project demonstrates practical implementation of:

- Simple Linear Regression
- Multiple Linear Regression
- Multicollinearity Analysis (VIF, OLS Summary)
- Logistic Regression
- Categorical Variables in Regression
- Polynomial (Quadratic) Regression

## Tools Used
- Python
- pandas
- numpy
- matplotlib
- scikit-learn
- statsmodels

## Datasets
- Butler Dataset
- Butler with Gas Consumption
- Butler Highway Dataset
- Reynolds Sales Dataset

## Key Concepts Demonstrated
- Least Squares Estimation
- R-squared Interpretation
- Residual Analysis
- Logistic Probability Modeling
- Odds Ratio Interpretation
- Multicollinearity Diagnosis
- Polynomial Feature Engineering

This project is part of advanced statistical modeling practice.
